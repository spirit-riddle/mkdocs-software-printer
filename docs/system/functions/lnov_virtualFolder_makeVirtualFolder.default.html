  • Virtual Folder AI Factory

    Creates a utility object for AI interactions related to virtual folder operations. Each method maintains a separate conversation context with the AI model.

    Available Methods:

    • getResponseFromPlanningAi: Interacts with the AI for planning tasks.
    • getResponseFromFileCompressionAi: Interacts with the AI for file compression tasks.
    • getResponseFromDebuggerAi: Interacts with the AI for debugging tasks.
