  • MkDocs Utility Factory

    Creates a utility object for interacting with MkDocs projects, providing methods to find, parse, and read MkDocs documentation files.

    Available Methods:

    • findMkDocsFiles: Finds MkDocs YAML configuration files in a directory recursively.
    • parseMkDocsYAML: Parses a MkDocs YAML configuration file.
    • readMarkdownFromNav: Reads markdown content based on the navigation section in the MkDocs configuration.
